## рџљЂ Run It On Your System

### 1. Prerequisites

- **Node.js 20+** ([install](https://nodejs.org/))
- **Bun** (recommended) or npm вЂ” install Bun: `curl -fsSL https://bun.sh/install | bash`
- A modern browser

### 2. Clone & install

```bash
git clone <your-repo-url> poseidon
cd poseidon
bun install
# or: npm install
```

### 3. Environment variables

The project uses **Lovable Cloud**, which auto-generates an `.env` file containing:

```bash
VITE_SUPABASE_URL="https://<your-project>.supabase.co"
VITE_SUPABASE_PUBLISHABLE_KEY="<anon-key>"
VITE_SUPABASE_PROJECT_ID="<id>"
SUPABASE_URL="..."
SUPABASE_PUBLISHABLE_KEY="..."
```

If you cloned this repo from GitHub (no `.env` present), create one with the values above. For server-side AI calls, also set:

```bash
LOVABLE_API_KEY="<your-lovable-api-key>"
```

> Lovable AI Gateway access is bundled with Lovable Cloud and requires no separate API key when running inside Lovable. If running locally **outside** Lovable, supply `LOVABLE_API_KEY` in `.env`.

### 4. Start the dev server

```bash
bun run dev
# or: npm run dev
```

Visit **http://localhost:8080** (or whatever Vite prints).

### 5. Build for production

```bash
bun run build
bun run preview
```

The build targets a Cloudflare Worker via `@cloudflare/vite-plugin` (see `wrangler.jsonc`).
