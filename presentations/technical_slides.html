Data Flow & State Management

  • Client-side: localStorage hooks (useProfile, useNFTs) for instant persistence and zero network latency.
  • Server-side: Supabase for durable storage and authentication.
  • Security Isolation: Gemini API keys and sensitive operations execute entirely within Vite's SSR layer via createServerFn. Keys never touch the client.
Script: "We use a dual approach: localStorage for instant client state, and SSR server functions to keep API keys completely isolated from the browser."
